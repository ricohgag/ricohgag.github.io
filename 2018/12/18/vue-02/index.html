#### 事件修饰符

  • .stop 阻止冒泡

  • .prevent 阻止默认事件（如a标签默认事件

  • .capture 添加时间侦听器使用事件捕获模式

  • .self 只当该元素本身(非子元素)触发时触发回调

  • .once 事件只触发一次

##### v-model和双向绑定

  • v-bind只能实现单向绑定，从 M 绑定到 V,无法实现数据的双向绑定。

  • 使用v-model指令能实现表单元素和 Model 中数据的双向绑定，只能运用在表单元素中表单元素中。如（input[radio，text，address，email…] select checkbox textarea。
